Line Sensor Calibration\n- Phase 1: Field calibration\n- Phase 2: Line verification (optional)\n- Manual adjustment with phase references\n- Live min/max display\n- Skip to manual mode\n\n### 2. Goal Color Calibration\n- Yellow and blue goal support\n- Interactive region drawing on live video\n- Manual HSV picker (3 canvas maps)\n- Automatic HSV calculation from regions\n- Region merging (min/max aggregation)\n\n### 3. Ball Color Calibration\n- Orange ball support\n- Same dual-method approach as goal colors\n- Interactive + manual modes\n- Auto-computation from drawn regions\n\n### 4. Camera Ball Distance\n- Single-point calibration\n- Known distance input\n- Live camera feed\n- Instant feedback\n\n### 5. Goal Distance Calibration\n- Two-distance setup (init + line)\n- Real-time phase tracking\n- Live video with detection overlay\n- Built-in mode control\n- Dynamic instructions\n\n### 6. Reset Compass\n- One-click operation\n- Positions current orientation as 0°\n- Auto-success feedback\n\n## How to Use\n\n### Opening the Calibration Menu\n\n```\nControl Panel → \"Open Calibration Menu\" button\n```\n\nThis opens a 2x3 grid of subsystem buttons. Click any to enter that subsystem.\n\n### Line Sensor Workflow\n\n```\n1. Click \"Line Sensors\"\n2. Choose:\n   a) \"Start Phase 1\" → Drive around field → \"Apply & Continue\" → (Choose: Phase 2 or Manual)\n   b) \"Start Phase 2\" → Drive across lines → \"Apply & Continue\" → Manual\n   c) \"Open Manual Adjustment\" → Edit 12-row table → \"Apply & Close\"\n3. Close modal\n```\n\n### Color Calibration Workflow\n\n```\n1. Click \"Goal Colors\" or \"Ball Color\"\n2. Select color (yellow/blue for goals)\n3. Choose method:\n   a) Draw regions on camera → \"Add Region\" (repeat) → Auto-fills HSV picker\n   b) Or manually edit HSV sliders/inputs\n4. \"Apply & Close\"\n```\n\n### Distance Calibration\n\n```\n1. Click \"Camera Ball Distance\" or \"Goal Distance\"\n2. Enter known distance(s)\n3. \"Start Calibration\"\n4. (For goal: move robot manually; for ball: just point camera)\n5. \"Stop & Calculate\"\n6. Modal auto-closes on success\n```\n\n### Reset Compass\n\n```\n1. Click \"Reset Compass\"\n2. Position robot (front facing north or reference)\n3. \"Reset Compass\"\n4. Auto-closes on success\n```\n\n## Configuration\n\nTo disable a feature, edit the respective modal:\n\n### Disable Interactive Region Drawing\nIn `src/components/calibration/subsystems/GoalColorCalibrationModal.tsx`:\n```tsx\nconst USE_INTERACTIVE_METHOD = false;  // Line 9\n```\n\n### Disable Manual HSV Picker\nIn `src/components/calibration/subsystems/GoalColorCalibrationModal.tsx`:\n```tsx\nconst USE_MANUAL_METHOD = false;  // Line 10\n```\n\n## Testing\n\n1. **Development**: `npm run dev` → http://localhost:3000\n2. **Build**: `npm run build` (should complete with no errors)\n3. **Production**: `npm start`\n\n## API Integration\n\nAll communication goes through `robotClient` in `src/lib/robotAPIClient.ts`.
