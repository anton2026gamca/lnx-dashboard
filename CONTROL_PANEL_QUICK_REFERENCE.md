color\n- `getMotorSettings()` - Fetch motor settings\n- `setMotorSettings(settings: MotorSettings)` - Update motor settings\n- `getAutonomousSettings()` - Fetch autonomous settings\n- `setAutonomousSettings(settings: AutonomousSettings)` - Update autonomous settings\n- `getAllStateMachines()` - Get available state machines\n- `setManualControl(angle, speed, rotate)` - Send manual control (from ManualMovementComponent)\n\n## Error Handling\n\nAll components include error handling:\n\n1. **Try-Catch Blocks**: API calls are wrapped in try-catch\n2. **Error Messages**: Errors are displayed to the user\n3. **State Revert**: Local state reverts to remote state on error\n4. **Graceful Fallbacks**: UI remains functional even with errors\n\n## Performance Optimizations\n\nвң… **No
